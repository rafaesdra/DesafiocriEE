# 🎮 Engenharia XP

Plataforma gamificada de desafios diários de matemática aplicada à Engenharia, 
com foco em ajudar alunos do primeiro período da UFCG a superar a etapa inicial do curso.

## 🎯 Problema

Grande parte dos alunos enfrenta dificuldades nas disciplinas matemáticas iniciais,
como Cálculo e Física, o que aumenta reprovações e evasão.

## 💡 Solução

O Engenharia XP propõe:

- Conteúdo resumido e didático
- Questões de múltipla escolha
- Explicação detalhada após resposta
- Desafios diários
- Sistema gamificado de progresso

## 🚀 Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- JSON (base de dados)
- Hospedagem via Vercel

## ▶️ Como Executar

1. Clone o repositório
2. Abra o arquivo `index.html`
3. Ou acesse online via Vercel

## 📌 Objetivo

Reduzir a dificuldade inicial enfrentada por estudantes de Engenharia,
tornando o aprendizado mais visual, interativo e motivador.
